# ide-golang
Go language support for [Atom-IDE](https://github.com/facebook-atom/atom-ide-ui). You should also install the [atom-ide-ui](https://atom.io/packages/atom-ide-ui) package to expose the functionality with Atom.

## Features

-   Code Completion
-   Code Format
-   Diagnostics
-   Document outline
-   Find references
-   Go to definition
-   Hover
